Programação Web

Na Unidade Curricular Programação Web abordaremos os princípios básicos associados ao desenvolvimento para a Web, tanto do lado back-end como front-end.

Aprender-se-ão as seguintes tecnologias 🌐:

  • HTML
  • CSS
  • Python
  • Django
  • JavaScript

De forma a consolidar todos os conhecimentos, será feito um laboratório semanal, de entraga obrigatória, de aplicação da matéria lecionada. Irá ser desenvolvido também um projeto.
